# 📖 المساعدة – بوت المواعيد

## الأوامر
| الأمر | الوظيفة |
|-------|---------|
| `/start` | القائمة الرئيسية |
| `/help` | هذا الدليل |
| `/cancel` | إلغاء الحجز |

## خطوات الحجز
1. اضغط **"حجز موعد جديد"**
2. اختر **الخدمة**
3. أدخل **عدد الأفراد** (1–10)
4. اختر **التاريخ** و **الوقت**
5. أدخل **الاسم والهاتف** (مرة واحدة فقط؛ ويتم تخطي هذه الخطوة للمستخدمين العائدين)
6. اضغط **"تأكيد"**

## مواعيدي
اضغط **"مواعيدي الحالية"** لعرض حجوزاتك القادمة. تبقى المواعيد السابقة ظاهرة لمدة
**14 يوماً** قبل حذفها تلقائياً.

## لوحة المشرف (للمشرفين فقط)
| الإجراء | الوصف |
|---------|-------|
| `/admin` | فتح لوحة التحكم |
| **حجوزات اليوم** | عرض مواعيد اليوم، مرقمة `1، 2، 3 …` لكل يوم |
| **الحجوزات القادمة** | عرض جميع المواعيد المستقبلية |
| **إلغاء** | عبر زر الإلغاء بجانب كل حجز |
| **استيراد خدمات** | رفع ملف `.xlsx` / `.csv` (العنوان، المدة، is_active) |

## سياسة الاحتفاظ
| النطاق | المدة |
|--------|-------|
| أرشيف المشرف / العام | 7 أيام |
| سجل المستخدم | 14 يوماً |
| التنظيف التلقائي | يومياً عند منتصف الليل |

## اللغة
تُضبط لغة البوت في ملف `.env` عبر `DEFAULT_LANGUAGE` (`en` أو `ar`).
